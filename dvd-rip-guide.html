    DVD Rip Guide

    Software Requirements: MakeMKV, Handbrake

    1. Insert your DVD into your disc drive.
    2. Open MakeMKV.
    3. Click on the large disc drive icon.
    4. On the left hand side, select the title with the largest file size, as this is typically the movie file.
    5. Select the output folder where you want to save the movie file.
    6. Click the Make MKV button to the right of the output folder.
    7. Close MakeMKV.
    8. Open Handbrake
    9. Select "File" and then navigate to the movie file you just saved.
    10. Now, in each of the tabs here in Handbrake, we're going to change some settings.
    11. In the Summary tab, change "Format" to MKV, and make sure the "Passthru Common Metadata" box is checked.
    12. In the Dimensions tab, make sure the "Resolution Limit" is set to "480p NTSC SD".
    13. In the Filters tab, you can turn everything off.
      • However, if your DVD has interlaced video (horizontal lines on movement), you'll want to set the "Deinterlace" setting to "Decomb". You can check for interlaced video by using the "Preview" option in Handbrake.
    14. In the Video tab, set "Video Encoder" to "H.265 (x265)", set "Framerate" to "Same as source", select "Avg Bitrate" and set it to "1500", turn off "Multi-Pass Encoding", set the "Encoder Preset" to "Veryfast", Encoder Tune to "None", Encoder Profile to "Main", and Encoder Level to "Auto".
    15. In the Audio tab, set "Codec" to AAC, Quality to "Bitrate" and "192", and "Mixdown" to "Stereo".
      • If you know there are other audio tracks you want to include, like a commentary track, you can select the "Tracks" dropdown menu, choose "Add new track" and then change it to the appropriate "Source", following the same settings for the last track otherwise.
    16. In the Subtitles tab, under the "Tracks" dropdown menu, you can add a new track, just make sure it's set to English, or whatever language(s) you want.
    17. In the Chapters tab, it should automatically have any chapters listed, just make sure the "Create chapter markers" box is checked.
    18. Now, in the bottom right, select "Browse" to choose our save destination, as well as the name of the file, which should then fill in next to the "Save As:" field.\
    19. Everything should be ready now, so hit the big green "Start Encode" button at the top.
    20. Once it's finished, congratulations, you just ripped your DVD! The file should be where we told it to save.
