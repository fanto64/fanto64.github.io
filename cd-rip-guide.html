CD Rip Guide

Published on: June 01, 2024

This guide assumes you've already followed my EAC Install Guide so if you haven't, go do that first, then come back here.

CD Rip Guide

  1. Open Exact Audio Copy.
  2. Insert the disc you want to rip into your drive.
  3. A window should popup and begin to search for Metadata. Let it do it's thing, and once it's done, select the correct version of your disc from the list and a cover image, and then hit OK.
  4. Make sure all of the tags are correct and how you want them. If it's a multi-disc album, make sure to set the correct Number of CDs and CD Number.
  5. Now, at the top of the window, head to Action -> Detect Gaps, and allow it to do it's thing detecting the gaps between the tracks.
  6. Now, it's time to prepare for the rip.
  7. Make sure you have an easy to access folder to store your rips, like "C:\Rips".
  8. Now, in EAC, head to Action -> Create CUE Sheet -> Multiple WAV Files With Gaps... (Noncompliant)
  9. Navigate to your C:\Rips folder, and create a new folder with the following naming convention, which you may recognize from my Vinyl Rip Guide:
    • Artist - Album (Original Release Year) [FLAC] {Re-release Year - Country of Origin - Record Label - Catalog Number}
    • So, using Dark Side of the Moon as our example again, it would look like: Pink Floyd – Dark Side of the Moon (1973) [FLAC] {Harvest – SMAS-11163}
    • Or, if it was a remastered version, it might look like this instead: Pink Floyd – Dark Side of the Moon (1973) [FLAC] {2016 Remastered – US – Pink Floyd Records – PFRLP8}
  10. Enter the newly created folder, and hit Save to save the cuesheet there.
  11. Now, we're ready to rip. Head to Action -> Test & Copy Selected Tracks -> Compressed. It should pop up with a browse window which already has your newly created folder from step 9, so just hit OK.
  12. And now we are ripping! Simply sit back and wait for the process to complete as it rips each track and then compresses it to FLAC.
  13. Once it's done, you can take the CD out of your drive, and if you have more discs to rip, simply repeat the whole process starting from step 2.
